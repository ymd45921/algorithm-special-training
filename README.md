|              OJ 全称              | 前缀 |                       解释说明                       |
| :-------------------------------: | :--: | :--------------------------------------------------: |
|           洛谷（Luogu）           |  lg  |              Luogu P4779 -> `lg4779.cc`              |
|            Codeforces             |  CF  | 对于 Div.1 + Div.2 的赛题，尽量使用 Div.2 的比赛编号 |
|          Codeforces Gym           | Gym  |                                                      |
| [Dark bzoj](https://darkbzoj.tk/) | dbz  |                                                      |
|             Libre OJ              | loj  |                                                      |

### 提交命名

总的来说没什么要求，用**英文**说明做了什么就可以。

### 其他要求

每个专题/板块下都要有一个 `README.md`，用来说明这个板块的内容以及一些笔记。
